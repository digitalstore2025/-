# QudsCast AI

منصة إعلامية عربية لأتمتة إنتاج الأخبار بالصوت والفيديو

## المميزات

- **تحويل الخبر إلى سكربت إذاعي** - كتابة نص إذاعي مهني باللغة العربية الفصحى
- **توليد صوت المذيع** - استنساخ صوت المذيع عماد نور باستخدام TTS
- **إنتاج MP3 إذاعي** - دمج الجينغل مع الصوت وتطبيق معالجة صوتية احترافية
- **إنتاج فيديو 9:16** - فيديو عمودي جاهز للنشر على السوشيال ميديا
- **واجهة مستخدم بسيطة** - تصميم عربي RTL سهل الاستخدام

## المتطلبات

- Node.js 18+
- FFmpeg
- Python 3.11+ (اختياري - للـ TTS)

## التشغيل على Replit

1. انسخ المشروع إلى Replit
2. اضغط على زر **Run**
3. انتظر تثبيت الاعتمادات
4. افتح الرابط في المتصفح

## التشغيل المحلي

```bash
# تثبيت الاعتمادات
npm run install:all

# تشغيل التطوير
npm run dev

# أو تشغيل الإنتاج
npm run build
npm run start
```

```
└── replit.nix            # اعتمادات Nix
```

## الاستخدام

1. أدخل نص الخبر في الحقل
2. اضغط على زر "إنتاج المحتوى"
3. انتظر اكتمال المعالجة
4. حمّل MP3 أو MP4 أو انسخ الكابشن

## ملاحظة قانونية

تم الحصول على موافقة خطية صريحة من المذيع عماد نور على استخدام صوته لأغراض إعلامية محددة.
